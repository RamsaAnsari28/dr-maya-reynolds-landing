## Project Overview

This project is a **responsive landing page** built as part of a front-end assignment.  
The website represents an **imaginary therapist**, Dr. Maya Reynolds, PsyD, and showcases her services, approach, office, and FAQs.

The goal of the assignment was to:
- Follow the given **layout and content structure**
- Implement the design using **Next.js and Tailwind CSS**
- Keep the content as provided (no creative changes)
- Ensure the site is **responsive and clean**
- Add proper **SEO meta tags**

---

## Sections Included

- **Hero Section** – Introduction and call-to-action  
- **About Section** – Profile of Dr. Maya Reynolds  
- **Services Section** – Anxiety, Trauma, and Burnout therapy  
- **Approach Section** – Therapy methodology and philosophy  
- **Office Section** – Office description with exactly two images  
- **FAQ Section** – Common client questions  
- **Footer** – Contact information and call-to-action  

All sections are implemented **exactly as per the assignment instructions** without changing the provided content.

---

## Tech Stack

- **Next.js (React)**
- **Tailwind CSS**
- **next/image** for optimized images
- **next/head** for SEO meta tags

---

## Features

- Fully responsive (mobile, tablet, desktop)
- Clean and professional layout
- SEO meta tags (title, description, Open Graph, Twitter Card)
- Optimized images using Next.js Image component
- Structured layout matching the assignment instructions
